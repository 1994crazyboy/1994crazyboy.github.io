Windows 下 使用gogs搭建gitzhi

1.gogs简介

gogs是一款基于go语言的自助Git服务，通过gogs就可以搭建公司企业内部的Github，除了gogs之外还有大名鼎鼎的Gitlab，但是gitlab搭建太过于麻烦了，虽然功能十分的丰富，但是还是难道了很多人。当然还有gitstack、gitblit、bonobo等等一些，网上比较热门的应该属于gogs了，毕竟国人还发的，还是支持的！

3.gogs安装

下载解压出来后的目录为：

在解压目录下打开cmd，然后在运行gogs.exe，建议不要双击运行 ！

这里提示出了运行gogs相关运行参数。这里提示参数web是开启gogs web服务。

然后运行==gogs.exe web==

运行成功后，会提示Listen：http://0.0.0.0:3000，这个就是gogs的服务地址，用浏览器打开即可弹出安装gogs提示界面。

ps:其实就是输入http://localhost:3000即可，首次运行会跳转到安装页面，按照提示即可完成安装。gogs支持的数据库有mysql postsql sqlite，这里使用的是mysql，数据库默认名称为gogs。所以我们也按照默认名称创建一个用于保存gogs服务相关信息的数据库。
