小程序之JavaScript

Posted on 2018-05-05

  1. 小程序JavaScript和EMCAScript

    • JavaScript是对EMCAScript的一种实现
    • 小程序JavaScript的框架示意： EMCAScript --> 小程序框架 --> 小程序API
    • 浏览器JavaScript的框架示意：EMCAScript --> DOM --> BOM
    • Nodejs的JavaScript框架示意：EMCAScript --> NPM --> Native
    • 小程序没有DOM所以依赖DOM的JQuery无法使用
    • IOS9和IOS10 不支持ES6，所以需要在IDE中将ES6语法转换为ES5语法
  2. ##Commonjs模块化
    • 小程序内部使用commonjs来进行模块的管理和处理，所以需要遵循commonjs规范
    • commonjs使用require方法加载模块
    • 模块加载是同步的，只有加载完成后才能执行后面的操作
    • module.exports属性是当前模块对外的接口，调用模块实际上是调用module.export的变量
